# Deema IO Personal Website

A clean static personal portfolio website for the GitHub username `deemaio`.

## Files

- `index.html` — website structure
- `styles.css` — responsive design and visual styling
- `script.js` — mobile navigation and current year
- `assets/favicon.svg` — simple favicon

## How to use with GitHub Pages

1. Create a public repository named exactly:

   `deemaio.github.io`

2. Upload all files from this folder to that repository.

3. Go to **Settings → Pages**.

4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Your website should become available at:

   `https://deemaio.github.io`

## What to edit

In `index.html`, update:

- About text
- Project cards
- Email address
- LinkedIn link
- GitHub profile link, if the username changes
